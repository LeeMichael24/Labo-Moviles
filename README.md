Proyectos de Android Studio con Kotlin
Este repositorio contiene una colección de proyectos de Android Studio desarrollados con Kotlin y utilizando la versión API 24 (Android 7.0 Nougat).

Requisitos
Android Studio (versión 3.0 o superior)
JDK 1.8 o superior
Dispositivo Android o emulador con API 24 o superior
Proyectos incluidos
Proyecto 1: Una aplicación sencilla que muestra una lista de tareas pendientes.
Proyecto 2: Una aplicación de notas que utiliza una base de datos local para almacenar las notas.
Proyecto 3: Una aplicación de clima que utiliza una API de terceros para mostrar la información del tiempo.
Cómo usar este repositorio
Clona este repositorio en tu máquina local.
Abre Android Studio y selecciona la opción de "Open an existing Android Studio project".
Navega hasta la carpeta donde clonaste este repositorio y selecciona uno de los proyectos.
Espera a que Android Studio cargue todas las dependencias y configura el emulador o dispositivo que vas a utilizar para probar la aplicación.
Haz clic en "Run" para compilar y ejecutar la aplicación en el emulador o dispositivo.
